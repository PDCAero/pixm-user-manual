# PIXM Development

### Data Model

The PIXM data model was developed with the industry standard Sparx Enterprise Architect software.  The current Sparx model file can be found here:

[PIXM Sparx EA Model](https://pixmaerolibrary.blob.core.windows.net/library-files/PIXM_snapshot15OCT25.qeax)

If you do not currently have Sparx EA installed, a free viewer can be found [here](https://sparxsystems.com/products/ea/downloads.html).


The data model in UML format is also expressed and navigable here:

[PIXM Data Model Overview](https://pixmaerolibrary.z20.web.core.windows.net/)


### XML Schema Definition Files

The current version of the PIXM Schema Definition files (XSD) can be found [here.](https://pixmaerolibrary.blob.core.windows.net/$web/PIXM Schema _ v0.1 _ 15OCT25.zip)
