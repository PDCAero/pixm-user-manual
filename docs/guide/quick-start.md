# Quick Start

Get up and running in minutes.

## Prerequisites

- A GitHub account
- Your repo set up with GitHub Pages pointing to the `docs/` folder on `main`

## Steps

1. Clone the repo
2. Edit `docs/_sidebar.md` to set up your navigation
3. Add your `.md` content files
4. Push to `main`

Your changes appear live at `https://YOUR-ORG.github.io/YOUR-REPO/` within seconds.
