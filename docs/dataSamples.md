# PIXM Data Samples

PIXM's initial efforts focus on takeoff and landing data structures derived primarily, though not exclusively, from the existing SCAP Takeoff and Landing specifications and elements of existing AIRM models.  The files provided below are intended to be provided as examples for what 'typical' PIXM data files might look like when being used in Takeoff and Landing performance calculation.

## Landing Files

[Landing Response](https://pixmaerolibrary.blob.core.windows.net/library-files/Examples/landingResponse.xml)
